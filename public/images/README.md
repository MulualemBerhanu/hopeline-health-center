# Images Directory

Please place your hero background image here with the filename: `hero-background.png`

The image should be:
- High resolution (recommended: 1920x1080 or larger)
- Format: PNG (or JPG, WebP)
- The image should show two people helping each other climb a mountain at sunset (as described in the requirements)

The image will be used as the background for the hero section on the homepage.
